# easy-quiz

Я хочу чтобы в этом пустом репозитории ты написал для меня с нуля веб-приложение.

Приложение - это сервис по быстрому созданию квизов, и отслеживанию прохождения студентами в реальном времени.

При этом важно - квиз может быть создан в любой день, но лектор должен получить свои qr-код и ссылку на него. Затем, в день лекции, qr-код выводится на экран, а студенты открывают его у себя и задают себе любой анонимный никнейм.

Лектор нажимает на кнопку у себя, и квиз начинается. При этом на экране выводится статистика по правильным или неправильным ответам (анонимно). Всё это происходит в реальном времени, а результаты экспортируются в csv файл для лектора (уже не анонимно).

Всё должно работать на базовых языках: html, css, php, javascript. В коде должен присутствовать index-файл, который предлагает создать свой квиз.
